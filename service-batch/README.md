
## 📌 개요
이슈모아(https://issuemoa.kr) 에 사용 될 각종 데이터를 수집(크롤링, API, 한국어 형태소)하여 적재(Mongo) 한다.

## 📋 배치 목록
- 뉴스 랭킹
- 유튜브 인기 급상승
- 키워드 생성 (화제 키워드)
